# Endee-Sodh: Semantic Search Engine

**Endee-Sodh** (meaning *Search* or *Discovery*) is a high-performance semantic search engine built using the **Endee.io** vector database. Unlike traditional keyword-based search, this project leverages natural language processing (NLP) to understand the underlying context and intent of a query, providing more accurate and relevant results.

---

## 🚀 About the Project

In the era of Big Data, retrieving information based on exact word matches is no longer sufficient. This project demonstrates a complete pipeline for **Semantic Retrieval**, where text data is transformed into high-dimensional mathematical vectors (embeddings) that represent their meaning.

### Key Features:
* **Contextual Understanding:** Uses the `all-MiniLM-L6-v2` transformer model to map sentences to a 384-dimensional vector space.
* **High-Speed Retrieval:** Utilizes **Endee.io**, a C++ based vector database optimized for modern CPU architectures (AVX2/AVX512), ensuring sub-millisecond search latency.
* **Scalable Architecture:** Designed with a RESTful API approach, allowing the search engine to be easily integrated into web applications or RAG (Retrieval-Augmented Generation) pipelines.

---

## 🛠️ How Endee.io is Utilized

**Endee.io** serves as the core storage and indexing layer of this project. It is responsible for managing the high-dimensional vectors generated by our embedding model.

### 1. Vector Indexing
Once the Python script processes raw text through the `SentenceTransformer`, the resulting numerical arrays (vectors) are pushed to the Endee server. Endee indexes these vectors using optimized data structures that allow for rapid traversal of the vector space.

### 2. Similarity Search (Nearest Neighbors)
When a user submits a query like *"Where is Goa?"*, Endee does not look for the word "Goa." Instead:
1.  The query is converted into a vector.
2.  Endee performs a **Cosine Similarity** or **Euclidean Distance** calculation.
3.  It retrieves the "Nearest Neighbors"—the stored vectors that are mathematically closest to the query vector—and returns the original text.

### 3. Local-First Performance
Because Endee is written in C++ and runs as a lightweight local service, it provides production-grade vector search capabilities without the overhead of cloud-based managed services, making it ideal for privacy-focused or edge-computing applications.
